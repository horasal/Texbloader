Texbloader
==========

Parse and covert KLab's Texb image.

## Description

Codes for reading/converting KLab's tiling image **texb**.
Description for texb can be found at KLab's open source engine [PlaygroundOSS](https://github.com/KLab/PlaygroundOSS)
